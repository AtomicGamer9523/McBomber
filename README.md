# McBomber

## Bomb Bases Easier

### Build

no enderpearl:

```bash
git clone https://github.com/AtomicGamer9523/McBomber
cd McBomber
cargo build --release
./gradlew build
```

enderpearl:

```bash
enderpearl stasis
cd McBomber
enderpearl
```

### Running

```bash
mv ./build/libs/mcbomber-0.1.jar MINECRAFT_MODS_FOLDER
cargo run --release
```

Make sure rust is running before starting up minecraft
